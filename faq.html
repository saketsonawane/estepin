No. Login is not required for booking a time slot.

  1. Click on eStepin → Token Booking.
  2. Click on Booking.
  3. Select District.
  4. Select Shift (by default General shift is selected).
  5. Select Booking Date.
  6. Select SRO Office.
  7. Click Continue. If booking is available then only booking is allowed
  8. Enter Public Data Entry Number (ISARITA/MKCL).
  9. Click Verify.
  10. Select preferred Time Slot. first half of each shift timeslots are reserved for epayment (eChallan(GRAS)/eSBTR)
  11. Click Book.
  12. After successful data entry system will show booking message(Message will contain SRO office name,time slot token ID and RebookingId assigned)
  13. Click on Reciept button for reciept printing(Use ctrl+P for recipt printing on printer).
  14. Note down your TokenID and RebookingId for further use.

Data Entry Number Generate through Public Data Entry (ISARITA) or (MKCL) Example :

  • ISARITA Data Entry Number (11 digits)
  • MKCL Data Entry Number (21 digits)
